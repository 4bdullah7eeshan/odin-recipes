# Odin Recipes

## About

This is a website that features recipes of dishes that lived through history.

This was created as part of [The Odin Project](https://www.theodinproject.com/)'s [Recipes](https://www.theodinproject.com/lessons/foundations-recipes) project.

## Learning Reflections

### Basics

- Setup folders and files for a web development project.

### HTML

- Write the basic HTML Boilerplate

#### Text

- Create various types of texts such as:
    - Paragraphs
    - Headings
    - Bold text
    - Emphasized text
    - Underlined text

#### Lists

- Create ordered lists
- Create unordered lists

#### Images

- Display images
- Resize images proportionally

#### Links

- Create links to other webpages of my own website
- Create links that open in a new tab/window

### CSS

- Add external CSS to HTML
- Change color of text elements
- Change background color of HTML elements
- Change font of an HTML text elements
- Align text horizontally
- Change the dimensions of images proportionally
