# Odin Recipes

This is a simple recipe website created with HTML.

## Project Structure

Here is a bird-eye view of the repo structure

```
├── LICENSE
├── README.md
├── index.html
└── recipes/
    ├── lasagna.html
    ├── lemonade.html
    └── pancake.html
```

## Getting Started

```bash
git clone git@github.com:4bdullah7eeshan/odin-recipes.git
cd odin-recipes
```
